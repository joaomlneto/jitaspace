Add `@jitaspace/solar-system-map`, a publishable React Three Fiber package that renders an interactive, schematic 3D map of a solar system — the star, planets (with their moons and asteroid belts), stations and stargates — with orbit controls, gentle auto-rotation and hover labels. It is presentational (props-driven, with a `renderLabel` hook) and has no data-fetching dependencies. The web solar-system page consumes it through a thin adapter that supplies ESI data and name-resolving labels.
