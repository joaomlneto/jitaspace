# @jitaspace/evekill-client

Auto-generated TypeScript client for the [EVE Kill](https://eve-kill.com) API.

## Overview

Provides typed axios functions and TanStack React Query hooks for the EVE Kill kill mail and statistics API. Generated from the EVE Kill OpenAPI spec using [Kubb](https://kubb.dev).

> **Do not edit files in `src/generated/` directly.** Run `pnpm kubb:generate` to regenerate.

## Usage

```ts
import { useGetKillmails } from "@jitaspace/evekill-client";
```

## Scripts

| Command | Description |
