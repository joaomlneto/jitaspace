# @jitaspace/utils

Shared TypeScript utilities for JitaSpace.

## Overview

A collection of pure utility functions with no external runtime dependencies, covering common patterns used across the JitaSpace monorepo.

## Exports

| Module | Description |
|---|---|
| ESI utils | Helpers for working with ESI responses and IDs |
| Fitting utils | EFT/DNA fitting string parsing and manipulation |
| Math utils | EVE-specific numeric helpers (ISK formatting, rounding) |
| Object utils | Typed object manipulation utilities |
| React utils | Small React-specific helpers |
| SDE utils | Static data lookup helpers |
| Skills utils | Skill point and level calculation functions |
| Type utils | TypeScript type guards and inference utilities |
| UI utils | Class name and style helpers |

## Usage

```ts
import { formatISK, parseEFT } from "@jitaspace/utils";
```
