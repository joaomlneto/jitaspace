# @jitaspace/fuzzworks-market-client

Auto-generated TypeScript client for the [Fuzzwork Enterprises](https://market.fuzzwork.co.uk) market API.

## Overview

Provides typed axios functions and TanStack React Query hooks for the Fuzzwork market data API (type prices, buy/sell aggregates). Generated from the Fuzzwork OpenAPI spec using [Kubb](https://kubb.dev).

> **Do not edit files in `src/generated/` directly.** Run `pnpm kubb:generate` to regenerate.

## Usage

```ts
import { useGetMarketPrices } from "@jitaspace/fuzzworks-market-client";
```
