# @jitaspace/evetycoon-client

Auto-generated TypeScript client for the [EVE Tycoon](https://evetycoon.com) market API.

## Overview

Provides typed axios functions and TanStack React Query hooks for the EVE Tycoon market data API. Generated from the EVE Tycoon OpenAPI spec using [Kubb](https://kubb.dev).

> **Do not edit files in `src/generated/` directly.** Run `pnpm kubb:generate` to regenerate.
>
> **Note:** This API has known CORS restrictions; use server-side calls where possible.

## Usage

```ts
import { useGetMarketStats } from "@jitaspace/evetycoon-client";
```
